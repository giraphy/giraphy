## Install giraphy

```
$ npm i -g giraphy-cli
$ giraphy init giraphy-example
$ cd giraphy-example
$ npm i 
cd giraphy-example
```

## Write your first giraphy.yaml
```
database:
  type: MySQL
  version: 5.7
  host: 127.0.0.1
  user: root
  password: root
  database: example
```

## Build your first schema-base.ts automatically
```
$ giraphy-cli rdbms build-schema
```
